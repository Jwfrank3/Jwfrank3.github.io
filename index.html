    on to the next topic. For some reason the html page is circling back to what different types of srcs there are. I've already covered absolute and relative code dependencies for sources and images yet here i am with more information regarding file system structure regarding that.



    file path structure

    path Description
    <'img src="picture.jpg"'> The "picture.jpg" file is located in the same folder as the current webpage.
    <'img src="images/picture.jpg"'> The "picture.jpg" file is located in the images folder in the current folder.
    <'img src="/images/picture.jpg"'> The "picture.jpg" file is located in the images folder at the root of the current web.
    <'img src="../picture.jpg"'> The "picture.jpg" file is located in the folder one level up from the current folder.
